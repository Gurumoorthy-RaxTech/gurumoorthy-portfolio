⚡ Part 24 Quick-Recall Cheat Sheet

  • CLR → runs the code (memory, GC, exceptions, JIT); BCL → the class library CLR ships with; CTS → common type system across .NET languages; CLS → the subset of rules all .NET languages must follow to interoperate
  • Managed vs Unmanaged → managed = CLR handles memory (GC); unmanaged = you manage memory yourself (C++/native)
  • GC Generations → Gen 0 (short-lived, collected often) → Gen 1 (buffer) → Gen 2 (long-lived, collected rarely, most expensive)
  • Middleware order → Exception Handling → HTTPS Redirect → Static Files → Routing → CORS → Authentication → Authorization → Endpoints
  • DI Lifetimes → Transient (every request new) → Scoped (per HTTP request) → Singleton (app lifetime, one instance)
  • Filter order → Authorization → Resource → Model Binding → Action → Exception (on error) → Result
  • ref vs out vs in → ref = must be initialized before passing; out = must be assigned inside method; in = read-only reference (no copy, no modify)
  • IEnumerable vs IQueryable → IEnumerable = in-memory, LINQ-to-Objects; IQueryable = translated to SQL, filters pushed to the database
  • GET vs POST vs PUT vs PATCH → GET = read, no body; POST = create, not idempotent; PUT = full replace, idempotent; PATCH = partial update
  • EF Core loading → Eager (.Include, one query with JOIN) → Lazy (auto-fetch on access, N+1 risk) → Explicit (.Load() manually when you decide)
  • Find() vs FirstOrDefault() → Find() checks the in-memory change tracker first (no DB hit if cached), FirstOrDefault() always queries
  • Any() vs Count() → Any() stops at the first match (fast); Count() enumerates everything (slow for just an existence check)

110. Reverse a string (Coding Problem)
Approach — char[] ku convert pannі Array.Reverse() use pannрроேн் (simplest), illainа manual ah two-pointer technique (start/end index, swap pannі convergе панна varaikkும்).
🗣️ Interview la eppadi sollanum"The quickest way is converting the string to a char array and calling Array.Reverse(), then constructing a new string from it. If asked to do it manually without built-ins, I'd use the two-pointer technique — swap characters from both ends moving toward the middle."
💻 Code (manual, no built-ins) 
static string ReverseString(string input)
{
    char[] chars = input.ToCharArray();
    int left = 0, right = chars.Length - 1;
    while (left < right)
    {
        (chars[left], chars[right]) = (chars[right], chars[left]);
        left++; right--;
    }
    return new string(chars);
}

    string s = new string(input.ToLower().Where(char.IsLetterOrDigit).ToArray());
    int left = 0, right = s.Length - 1;
    while (left < right)
    {
        if (s[left] != s[right]) return false;
        left++; right--;
    }
    return true;
}

static int? SecondLargest(int[] arr)
{
    if (arr.Length < 2) return null;
    int largest = int.MinValue, second = int.MinValue;
    foreach (var n in arr)
    {
        if (n > largest) { second = largest; largest = n; }
        else if (n > second && n < largest) { second = n; }
    }
    return second == int.MinValue ? null : second;
}

static List<int> RemoveDuplicates(int[] arr)
{
    var seen = new HashSet<int>();
    var result = new List<int>();

    }
    return result;
}

static void SelectionSort(List<int> list)
{
    for (int i = 0; i < list.Count - 1; i++)
    {
        int minIndex = i;
        for (int j = i + 1; j < list.Count; j++)
        {
            if (list[j] < list[minIndex]) minIndex = j;
        }
        (list[i], list[minIndex]) = (list[minIndex], list[i]);
    }
}
