Query eppadi execute aagum nu kaatura plan — table scan vs index seek, cost identify panna.
🗣️ Answer"An execution plan shows how SQL Server processes a query — indexes used, join types, estimated costs — helping find bottlenecks like table scans." Example: Table Scan therinjuchu, missing index add pannina Index Seek ku maarichu.

81. Flexbox?

🗣️ Answer"Flexbox is a one-dimensional CSS layout model for arranging items in a row or column with easy alignment." Example: display:flex; justify-content:space-between; for navbar.
82. Grid?
2-dimensional layout (rows AND columns), complex page layouts ku.
🗣️ Answer"CSS Grid arranges items in both rows and columns simultaneously — ideal for complex page layouts." Example: display:grid; grid-template-columns:250px 1fr; for a dashboard sidebar + content.
83. CSS Position?
static (default), relative (own position reference), absolute (nearest positioned ancestor), fixed (viewport, scroll aana move aagathu), sticky (relative → fixed on scroll).
🗣️ Answer"static is normal flow, relative offsets from itself, absolute positions relative to nearest positioned ancestor, fixed stays fixed to viewport, sticky toggles between relative and fixed on scroll." Example: position:sticky; top:0; for a header.
84. Event Bubbling?
Child element la event fire aana, parent elements ku um propagate aagum (inside-out).
🗣️ Answer"Event bubbling means an event on a child propagates upward through its ancestors in the DOM." Example: Button click → button → div → body, each level listener trigger aagum.
85. Event Delegation?
Parent mela oray listener vechu child events ah handle pannuradhu (bubbling use panni) — dynamic elements ku useful.
🗣️ Answer"Event delegation attaches one listener to a parent and uses bubbling to handle events on children — even dynamically added ones — more efficient than per-element binding." Example: $(document).on('click', '.delete-btn', fn) — future dynamic rows kооdа work pannum.
86. Closures?
Inner function outer function oda variables ah access pannikkonде irukkum, outer execution mudinja appuram kооdа.

87. AJAX?
Full page reload panna vendaam, server oda background la data exchange.
🗣️ Answer"AJAX allows asynchronous communication with the server without reloading the page, updating parts of the UI dynamically." Example: $.ajax({url:'/api/students',type:'GET',success:...})
88. Difference between JavaScript and jQuery?
JavaScript — core language. jQuery — library on top, DOM manipulation/AJAX easy panra wrapper.
🗣️ Answer"JavaScript is the core scripting language. jQuery is a library built on it that simplifies DOM manipulation, events, and AJAX with shorter cross-browser syntax." Example: document.getElementById('x').style.display='none'; vs $('#x').hide();
27G Microservices (Q89-94)
89. Monolithic vs Microservices?
Monolithic — oray codebase, oray deployment. Microservices — independent small services, own DB, independent deploy, API Gateway mூlaம communicate.
🗣️ Answer"A monolith is built and deployed as one unit. Microservices break the app into small, independently deployable services with their own databases, communicating over APIs." Example: Zen Campus la Student/Bill/Attendance/Hostel Services separate, Ocelot API Gateway mூlaம route pannuvom.
90. API Gateway?
Ella microservices ku um single entry point — routing, auth, rate limiting centralize ah handle pannum.
🗣️ Answer"An API Gateway is a single entry point routing client requests to the right microservice, while centrally handling auth, rate limiting, and logging." Example: Ocelot Gateway — client /gateway/students ku request anuppa Student Service ku route pannum.
91. Service Discovery?
Services dynamic ah scale/IP change aagum podhu, oru service veru service ah eppadi kandupidikkum nu mechanism.
🗣️ Answer"Service discovery lets microservices dynamically find each other's location without hardcoding addresses, especially as instances scale up/down." Example: Kubernetes DNS use panni Bill Service Student Service ah kandupidikkum.
92. Circuit Breaker?
Oru service down aana repeated calls pannama, "circuit open" panni fail fast aagi fallback kudukkura pattern.
🗣️ Answer"Circuit Breaker stops repeated calls to a failing dependency after a failure threshold, failing fast or returning a fallback to avoid cascading failures." Example: Payment Service down aana Polly circuit breaker calls block pannitu immediate "unavailable" message return pannum.
93. Messaging Queue?
Services asynchronous ah communicate panna use pannura queue (RabbitMQ/Azure Service Bus) — decoupled.
🗣️ Answer"A messaging queue enables async, decoupled communication — one service publishes, another consumes independently, improving resilience." Example: Payment success → "PaymentCompleted" event queue ku publish, Notification Service consume panni SMS anuppum.
94. Database per Service?
Each microservice own DB, veru service DB ah direct ah access panna kооdadhу.
🗣️ Answer"Each microservice owns its own database — no other service accesses it directly; communication happens only through APIs, ensuring loose coupling." Example: Bill Service ku student name venumna Student Service API call pannum, direct DB query pannathu.

🗣️ Answer"Merge combines two branches with a new merge commit, preserving both histories. Rebase replays your commits on top of another branch, producing a linear history without a merge commit." Example: Feature branch la git rebase main pannitu clean history vechu PR raise.
